# Projeto: Bushido School
## Visão Geral

O Bushido School é uma plataforma educacional projetada para facilitar a comunicação e interação entre alunos, professores e administração de uma escola. A plataforma oferece funcionalidades específicas para cada tipo de usuário, incluindo áreas de login distintas e recursos dedicados para matrículas, eventos, calendário e mais.
## Funcionalidades Principais
### Áreas de Usuários

#### Área do Aluno
Permite que os alunos acessem informações pessoais, como notas, atividades e calendário escolar.
Recursos adicionais incluem acesso ao chat interativo com professores, mural de notas e mural de atividades.

#### Área do Professor
Oferece aos professores recursos para gerenciar notas, atividades e comunicações com os alunos.
Recursos adicionais incluem acesso ao chat interativo com alunos e administração, além do registro de notas no mural dos alunos.

#### Área de Administração (Adm)
Permite à administração gerenciar matrículas, eventos, calendário e usuários.
Recursos adicionais incluem registro de novos professores e alunos, acesso ao chat interativo e gerenciamento de informações gerais da escola.
## Sobre as abas
#### Menu de Notícias

- <b>Notícias</b>: Apresenta todas as notícias relevantes para a comunidade escolar.
- <b>Eventos</b>: Informa sobre os eventos atuais e futuros da escola.
- <b>Programações</b>: Apresenta uma lista de futuros eventos e atividades planejadas.

#### Matrículas

- <b>Matrícula</b>: Permite que novos alunos se matriculem na escola.
- <b>Confirmar Matrículas</b>: Permite que os pais dos alunos, confirme a matrículas após a analise da administração. 
  - Administração recebe a solicitação de matrícula, faz a pré-matrícula e envia um código para o usuário que o recebe como uma mensagem em seu perfil. Esse código deve ser utilizado em um input na aba Confirmar Matrícula. Tudo dando certo, o usuário recebe uma mensagem dizendo que foi confirmada com sucesso e passa a ter acesso a todos os recursos do perfil(role) para os aluno.
- <b>Documentos</b>: Facilita o envio de documentos necessários para o processo de matrícula, pendentes ou para outras finalidades: permissão para viagem, acampamentos, etc.

#### Calendário

- <b>Calendário</b>: Apresenta um calendário com todas as programações escolares.
- <b>Ano Letivo</b>: Informa sobre as atividades e avaliações planejadas para o ano letivo.
- <b>Avaliações</b>: Lista as datas e tipos de avaliações a serem realizadas.
- <b>Eventos</b>: Mostra as datas dos eventos próximos da escola.
- <b>Programações</b>: Lista todas as programações por suas datas.

## Próximas Features

- <b>Chat Interativo</b>: Implementação de um chat interativo entre alunos, professores e administração.
- <b>Mural de Notas</b>: Exibição das notas dos alunos em seus perfis individuais.
- <b>Mural de Atividades</b>: Lista as atividades realizadas pelos alunos e nforma sobre as atividades e avaliações pendentes.
- <b>Mural de Faltas e Observações</b>: Registro das faltas e observações relevantes.
